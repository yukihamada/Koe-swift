# Koe iOS — プロダクトロードマップ

## 核心的な気づき

**iPhoneは既にKoe COINデバイス。**

マイク、スピーカー、WiFi、GPS、加速度センサー、画面 — 全部入ってる。
ハードウェアを売る前に、アプリだけでSolunaメッシュを動かせる。

- 無料アプリで10,000人がフェスに参加 → ネットワーク効果を証明
- もっと良い音・小さいデバイスがほしい人が $65のCOINを買う
- イベント主催者が $1,200のSUB / $1,500のFILLを買う

**アプリがフリーミアム。ハードウェアがプレミアム。**

## 3つのモード

### Mode 1: Koe (AI声アシスタント) — 今ある
- 音声認識 (Whisper ローカル)
- リアルタイム翻訳 (→英語)
- Mac Remote Whisper
- 履歴、テキスト展開、LLM後処理
- キーボード、ウィジェット、Live Activity

### Mode 2: Soluna (P2Pメッシュ) — 新規追加すべき
- 同じWiFiの他のKoeアプリ/デバイスと自動接続
- UDP multicast 239.42.42.1:4242 (音声) / 4243 (LED/画面同期)
- iPhoneのスピーカーから同期再生
- 画面全体がLED代わりに光る (sync.html をネイティブ化)
- マイクから群衆オーケストラに参加 (ピッチ検出→ハーモニー)
- チャンネル選択 (複数ステージ対応)

### Mode 3: Sound Memory (音の記憶) — 新規追加すべき
- バックグラウンドで常時録音 (リングバッファ、ローカル保存)
- 「さっき何て言った？」→ 音声検索で該当箇所を再生
- ダブルタップでブックマーク
- AIデイリーダイジェスト (1日を2分に要約)
- 録音データは端末内のみ (クラウドに送らない)

## 優先順位

### P0: 今すぐ (次のリリース)
1. **Solunaモード追加** — UDPマルチキャスト送受信 + 画面LED同期
   - 実装量: RecordingManagerにUDPソケット追加 + sync.html相当のSwiftUI画面
   - これだけでフェスのデモができる

2. **会話翻訳モード** — 2人が交互に話すと、互いの言語に翻訳
   - 日本語→英語 / 英語→日本語 を自動切替
   - 画面を2分割: 上に自分の言語、下に相手の言語

### P1: 1ヶ月以内
3. **Sound Memory (常時録音)** — バックグラウンド録音 + ローカル保存
   - 設定でON/OFF
   - 録音データは端末内のみ
   - 「検索」で音声をテキスト検索

4. **ギター入力対応** — Lightning/USB-C オーディオI/F からの入力
   - CoreAudioのルーティング設定追加
   - Solunaモードで他デバイスに配信

### P2: 3ヶ月以内
5. **Crowd Orchestra** — ピッチ検出 + ハーモニー参加
   - AccelerateフレームワークでFFT → ピッチ抽出
   - STAGEにピッチデータを送信
   - STAGEが生成したハーモニーを再生

6. **空間音声録音** — GPS + ジャイロで空間メタデータ付き録音
   - 後から3Dで再生可能

### P3: 将来
7. **オフラインAI応答** — ローカルLLM (Gemma 2B等) で完全オフライン
8. **音声感情分析** — 声のトーンからストレス/疲労を検出
9. **Apple Watch対応** — 腕からSoluna参加

## iPhoneアプリ vs COINハードウェア

| 機能 | iPhone (無料) | COIN ($65) |
|------|-------------|-----------|
| 音声認識 | ✅ (Whisper) | ✅ |
| 翻訳 | ✅ | ✅ |
| Solunaメッシュ | ✅ | ✅ |
| 画面LED同期 | ✅ (画面) | ✅ (WS2812B) |
| 群衆オーケストラ | ✅ | ✅ |
| 常時装着 | ❌ (ポケットに入れる) | ✅ (ペンダント/クリップ) |
| バッテリー影響 | ❌ (iPhoneの電池を消費) | ✅ (専用バッテリー) |
| イヤホン出力 | ✅ (既存) | ✅ (3.5mm) |
| サイズ | 大きい | 26mm |
| 音質 (スピーカー) | iPhoneのスピーカー | 20mmドライバー |

**80%の機能がアプリだけでカバーできる。** COINは「常時装着」「バッテリー独立」「小ささ」のプレミアム。
